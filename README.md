## How to run it?

```bash
# Fedora:
$ sudo dnf install chicken

# macOS with Homebrew:
$ brew install chicken

# executing a single script:
$ csi -s [filename]
```

See CHICKEN wiki:
* ["Platform packages"](https://wiki.call-cc.org/platforms)
* ["Using the interpreter"](https://wiki.call-cc.org/man/4/Using%20the%20interpreter)
